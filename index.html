Help - Commands & Features

Slash Commands

  • /me [action] - Perform an action (e.g., /me dances)
  • /nick [name] - Change username (guests only)
  • /color #RRGGBB - Change text color (e.g., /color #FF0000)
  • /clear - Clear your chat history
  • /status [status] - Set status (Online, Away, or Busy)
  • /join [room] - Join a room (general, random, images, windows)
  • /msg [user] [text] - Send private message to user
  • /help - Show this help window

Image Keywords

Type these to instantly send images:

  • !dog - Cute dog image
  • !cat - Cat image
  • !lol - Laughing face
  • !windows - Windows 95 logo
  • !error - Error icon
  • !cool - Cool face
  • !fire - Fire image

Features

  • Multi-room chat (4 different rooms)
  • Private messaging between users
  • Message reactions (thumbs up)
  • Image uploads with auto-pixelation
  • Achievement system (earn badges!)
  • Account system (save your progress)
  • Message history (see past messages)
  • @mention highlights (tag other users)
  • CRT mode (authentic retro effect)
  • Fully draggable windows

Tips

  • Press Enter to send messages quickly
  • Drag windows by their title bar
  • Click on images to view them full-size
  • Type @ followed by a username to mention them
  • Registered accounts save achievements automatically
